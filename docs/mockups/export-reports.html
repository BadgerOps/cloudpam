API Export Endpoint
For programmatic access and integration with other tools
Current API endpoint:
GET /api/v1/export?datasets=accounts,pools,blocks
Query Parameters:
datasets accounts, pools, blocks (comma-separated) format csv, json, xlsx (default: csv in zip) account_id Filter to specific account pool_id Filter to specific pool and children

Implementation Notes

  • Current API endpoint exists at /api/v1/export
  • Consider adding Excel (.xlsx) format support
  • PDF reports require server-side rendering (wkhtmltopdf or similar)
  • Large exports should be async with progress polling
  • Scheduled reports need a job scheduler (cron, Cloud Scheduler)
