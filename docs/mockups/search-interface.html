Implementation Notes

  • Performance target: <2 seconds for hundreds of pools
  • Results grouped by type (Pools, Hosts, Accounts)
  • Keyboard navigation fully supported
  • Debounce search input (300ms) to reduce API calls
  • Consider indexed search (e.g., SQLite FTS5) for scale
