CloudPAM UI Mockups - Utilization Display Options

Review each option and consider which would work best for different contexts. Multiple options can be combined (e.g., Option B for lists + Option C for detail view).

Option A: Inline Percentage Bar
Best for: Pool detail views, quick utilization check at a glance
prod-vpc-primary (10.0.0.0/16)

Pros

  • Immediately clear utilization
  • Shows exact numbers
  • Familiar pattern

Cons

  • Takes vertical space
  • Not great for lists
  • No trend information

Best for: Scanning lists, triage, identifying problems quickly

  • Compact, scannable
  • Shows hierarchy
  • Color draws eye to problems
  • Sortable columns

Cons

  • Less precise (just %)
  • No visual proportion
  • Need to set threshold config

Sparklines show utilization trend over past 30 days. Steep upward trends may indicate capacity planning needed.

Pros

  • Shows growth patterns
  • Predictive insight
  • Compact with rich data
  • Great for capacity planning

Cons

  • Requires historical data
  • More complex to implement
  • May be overkill for static pools

Recommendation: Combine Multiple Options

Based on different use cases, consider this combination:

  • Dashboard/Home: Option D (Heatmap) for quick overview
  • Pool List: Option B (Table with badges) for scanning and triage
  • Pool Detail: Option A (Bar) + Option C (Block map) for planning
  • Capacity Planning view: Option E (Sparklines) for trend analysis
